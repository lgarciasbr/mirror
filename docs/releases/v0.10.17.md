# v0.10.17 — Update Notice Tone

**Date:** 2026-05-25

## Highlights

- New-version welcome notices now use the ✨ icon.
- Pi renders the welcome notification as informational instead of warning.
- Update availability remains visible without looking like an error or problem.

## Why This Patch Exists

Production validation showed the release-awareness loop working end to end. The
remaining visual issue was tone: Pi rendered the welcome as a yellow warning
whenever a new version was available.

An available update is useful information, not a problem. It should attract
attention without implying danger.

## What Changed

The welcome update line now renders like:

```text
✨ New Version Available: v0.10.17
Ask: "what's new in this update?" or "update my Mirror"
```

The Pi extension now sends the welcome notification with `info` severity even
when the welcome includes an available update.

## Next Horizon

The release-awareness loop is now functionally validated and visually calmer.
Future polish can focus on richer UI affordances if Pi exposes actionable
notification buttons or links.
