# v0.10.13 — Pending Notes Ref Refresh Validation Stub

## Why This Patch Exists

remote-tracking refs before reading release notes. To validate that behavior like
a final user, production must already be on v0.10.12 and a newer stable release
must exist.

Runtime behavior is unchanged.
